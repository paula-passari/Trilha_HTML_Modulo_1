Estrutura Básica do HTML

Os elementos HTML são construídos usando tags, geralmente, uma tag de abertura, o conteúdo e uma tag de fechamento. Há algumas tags que não possuem fechamento. Um documento HTML deve conter as seguintes tags para o correto funcionamento nos navegadores:

<!DOCTYPE>

No topo de um documento HTML, colocamos o <!DOCTYPE> que informa aos navegadores a versão do HTML que está sendo usada. Isso ajuda o navegador a exibir as informações corretamente.

<html>

Informa ao navegador que o que está sendo carregado é um arquivo HTML. Ele é a raiz do documento e todas as outras tags devem ir entre suas tags.

<head>

É colocada dentro da tag <html>. Contém toda as informações do documento: título, o conjunto de caracteres que será usado no nosso documento, informações de estilo (CSS), etc. Tudo que estiver dentro dessa tag não é exibido pelo navegador.

<body>

É colocada dentro da tag <html> depois do fechamento da tag <head>. É o corpo do documento onde é incluído o conteúdo da página, ou seja, tudo que estiver dentro dessa tag será exbida pelo navegador.

Atributos

Os atributos são usados para definir certos comportamentos de um elemento HTML. Veja alguns exemplos de tags e atributos:

<a>

Define um hyperlink, ou seja, faz uma ligação entre as páginas. As ligações podem ser externa (do seu website para outro website); interna (as páginas que existem no seu website); e de seções (ligar as seções de uma única página). Pode receber alguns atributos, dentre eles, o mais importante é o atributo href que indica a página de destino. Outro atributo é o target que define se a página de destino será aberta em uma nova aba do navegador.
Clique aqui e veja outros atributos para essa tag


<img>

Inclui uma imagem na página HTML. Pode receber vários atributos, dentre eles, os mais importantes são o src que define o caminho da imagem e o alt que define um texto alternativo, se por algum motivo a imagem não puder ser exibida.
Imagem incluída utilizando a tag <img>:


<blockquote>

Define uma seção que é citada de outra fonte. Geralmente, o navegador faz uma indentação desse tipo de texto. O atributo a ser usado com essa tag é o cite que define a fonte da citação.
Veja uma citação do site WWF (World Wild Life):

For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.


<abbr>

Define uma descrição para uma abreviação. Use o atributo title para definir a descrição completa da abreviação que será exibida ao passar o mouse por cima da abreviação.
The WW2 was a global war that lasted from 1939 to 1945.

Listas

É possível fazer listas em um documento HTML. Geralmente, usa-se listas para fazer o menu de um website.
Os dois tipos de listas mais usados são as listas ordenadas e as listas desordenadas.

Lista Ordenada

Como o próprio nome diz, é uma lista de elementos ordenados. Para criá-la, usa-se a tag <ol> e usa-se a tag <li> para definir os itens da lista.
Veja um exemplo de lista ordenada:

  1. HTML
  2. CSS
  3. JavaScript


Lista Desordenada

Cria uma lista desordenada. Para criá-la, usa-se a tag <ul> e assim como a lista ordenada, usa-se a tag <li> para definir os itens.
Veja um exemplo:

  • HTML
  • CSS
  • JavaScript
